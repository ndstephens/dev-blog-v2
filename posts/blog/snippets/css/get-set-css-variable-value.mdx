I always forget how to do this and need to look it up.

## Get

```js
const primaryColorValue = getComputedStyle(
  document.documentElement
).getPropertyValue('--color-primary');
```

## Set

```js
document.documentElement.style.setProperty(
  '--color-primary',
  'hsl(230deg 100% 67%)'
);
```
